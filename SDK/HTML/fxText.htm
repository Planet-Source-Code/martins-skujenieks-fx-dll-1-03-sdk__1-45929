fxText

Function writes a character string with specified color and alignment at the specified location, using the currently selected font.

 

Declaration

Declare Function fxText Lib "fx.dll" ( _

ByVal DC As Long, _

ByVal X As Long, _

ByVal Y As Long, _

ByVal Text As String, _

ByVal Color As Long, _

ByVal Alignment As Long, _

ByVal Flags As Long) As Long

Parameters

DC

Identifies device context.

X

Specifies the logical x-coordinate of the reference point that will be used to align the string.

Y

Specifies the logical y-coordinate of the reference point that will be used to align the string.

Text

Specifies the text to write.

Color

Specifies the text RGB color.

Alignment

Specifies alignment code.

Flags

Reserved parameter.

Return Values

If the function succeeds, the return value is nonzero; if fails, the return value is zero.
