fxZoom

Zooms bitmap by specified value in or out.

 

Declaration

Declare Function fxZoom Lib "fx.dll" ( _

ByVal DestDC As Long, _

ByVal CenterX As Long, _

ByVal CenterY As Long, _

ByVal SrcDC As Long, _

ByVal SrcX As Long, _

ByVal SrcY As Long, _

ByVal SrcWidth As Long, _

ByVal SrcHeight As Long, _

ByVal Zoom As Double, _

ByVal TransparentColor As Long, _

ByVal Flags As Long) As Long

Parameters

DestDC

Identifies the destination device context.

CenterX

Specifies the x-coordinate, in logical units, of the center of the destination rectangle.

CenterY

Specifies the y-coordinate, in logical units, of the center of the destination rectangle.

SrcDC

Identifies the source device context.

SrcX

Specifies the x-coordinate, in logical units, of the upper-left corner of the source rectangle.

SrcY

Specifies the y-coordinate, in logical units, of the upper-left corner of the source rectangle.

SrcWidth

Specifies the width, in logical units, of the source rectangle.

SrcHeight

Specifies the height, in logical units, of the source rectangle.

Zoom

Specifies zoom value (0.00-16.00).

TransparentColor

The RGB color in the source device context to treat as transparent; if TransparentColor = -1, the transparency will not be used.

Flags

Reserved parameter.

Return Values

If the function succeeds, the return value is nonzero; if fails, the return value is zero.
