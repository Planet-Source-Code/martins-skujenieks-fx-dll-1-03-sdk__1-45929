fxDrawText

Function draws a character string with specified color and blending value to the specified rectangle, using given font object.

 

Declaration

Declare Function fxDrawText Lib "fx.dll" ( _

ByVal DC As Long, _

ByVal X As Long, _

ByVal Y As Long, _

ByVal Width As Long, _

ByVal Height As Long, _

ByVal Text As String, _

ByVal Color As Long, _

ByVal Font As Long, _

ByVal Blending As long, _

ByVal Flags As Long) As Long

Parameters

DC

Identifies device context.

X

Specifies the logical x-coordinate of the left top corner of the bounding rectangle.

Y

Specifies the logical y-coordinate of the left top corner of the bounding rectangle.

Width

Specifies, in logical units, the width of the bounding rectangle.

Height

Specifies, in logical units, the height of the bounding rectangle.

Text

Specifies the text to draw.

Color

Specifies the text RGB color.

Font

Identifies font object. Font object can be created with CreateFont API function.

Blending

Specifies blending value (0-255).

Flags

Reserved parameter.

Return Values

If the function succeeds, the return value is nonzero; if fails, the return value is zero.
