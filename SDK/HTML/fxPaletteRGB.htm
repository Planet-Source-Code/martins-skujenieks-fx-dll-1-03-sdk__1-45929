fxPaletteRGB

Macro returns a palette-relative RGB color specifier. An application using a color palette can pass this specifier, instead of an explicit RGB value, to functions that expect a color.

 

Declaration

Declare Function fxPaletteRGB Lib "fx.dll" ( _

ByVal R As Long, _

ByVal G As Long, _

ByVal B As Long) As Long

Parameters

· R

Specifies the intensity of the red color field (0-255).

· G

Specifies the intensity of the green color field (0-255).

· B

Specifies the intensity of the blue color field (0-255).

Return Values

The return value is a palette-relative RGB color specifier.
