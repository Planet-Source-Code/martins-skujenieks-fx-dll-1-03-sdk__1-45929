fxCMYK

Function calculates and returns a CMYK color value by combining the specified cyan, magenta, yellow, and black values.

 

Declaration

Declare Function fxCMYK Lib "fx.dll" ( _

ByVal C As Long, _

ByVal M As Long, _

ByVal Y As Long, _

ByVal K As Long) As Long

Parameters

C

Specifies cyan value (0-255).

M

Specifies magenta value (0-255).

Y

Specifies yellow value (0-255).

K

Specifies black value (0-255).

Return Values

The return value is the resulting CMYK color.
