fxAntiAlias

Displays anti-aliased bitmap, first stretching or compressing it to fit the dimensions of the destination rectangle, if necessary.

 

Declaration

ByVal DestDC As Long, _

ByVal DestX As Long, _

ByVal DestY As Long, _

ByVal DestWidth As Long, _

ByVal DestHeight As Long, _

ByVal SrcDC As Long, _

ByVal SrcX As Long, _

ByVal SrcY As Long, _

ByVal SrcWidth As Long, _

ByVal SrcHeight As Long, _

ByVal Flags As Long) As Long

Parameters

DestDC

Identifies the destination device context.

DestX

Specifies the x-coordinate, in logical units, of the upper-left corner of the destination rectangle.

DestY

Specifies the y-coordinate, in logical units, of the upper-left corner of the destination rectangle.

DestWidth

Specifies the width, in logical units, of the destination rectangle.

DestHeight

Specifies the height, in logical units, of the destination rectangle.

SrcDC

Identifies the source device context.

SrcX

Specifies the x-coordinate, in logical units, of the upper-left corner of the source rectangle.

SrcY

Specifies the y-coordinate, in logical units, of the upper-left corner of the source rectangle.

SrcWidth

Specifies the width, in logical units, of the source rectangle.

SrcHeight

Specifies the height, in logical units, of the source rectangle.

Flags

Reserved parameter.

Return Values

If the function succeeds, the return value is nonzero; if fails, the return value is zero.
