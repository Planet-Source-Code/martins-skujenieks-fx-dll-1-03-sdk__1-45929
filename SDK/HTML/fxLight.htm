fxLight

Renders light source on the bitmap.

 

Declaration

Declare Function fxLight Lib "fx.dll" ( _

ByVal DC As Long, _

ByVal X As Long, _

ByVal Y As Long, _

ByVal Color As Long, _

ByVal Extension As Long, _

ByVal Definition As Long, _

ByVal Flags As Long) As Long

Parameters

DC

Identifies the device context.

X

Specifies the x-coordinate, in logical units, of the light source.

Y

Specifies the y-coordinate, in logical units, of the light source.

Color

Specifies the light RGB color.

Extension

Specifies the light extension.

Definition

Specifies the light definition value (0-255).

Flags

Specifies light type:

  &H0  default

  &H1

Return Values

If the function succeeds, the return value is nonzero; if fails, the return value is zero.
