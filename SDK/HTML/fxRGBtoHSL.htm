fxRGBtoHSL

Function calculates and returns the hue, saturation and luminiscency values from specified RGB color.

 

Declaration

Declare Function fxRGBtoHSL Lib "fx.dll" ( _

ByVal RGB As Long, _

ByRef H As Double, _

ByRef S As Double, _

ByRef L As Double) As Long

Parameters

RGB

Specifies the RGB color.

H

Resulting hue value (0.00-1.00).

S

Resulting saturation value (0.00-1.00).

L

Resulting luminiscency value (0.00-1.00).

Return Values

Resulting hue, saturation and luminiscency values are applyed to referenced H, S, L variables.
